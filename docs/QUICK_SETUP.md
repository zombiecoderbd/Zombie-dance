# Quick Setup Guide

## 30-Second Setup (Linux/Mac)

```bash
git clone https://github.com/zombiecoder1/TypeScript-extensions-VS-Code.git
cd TypeScript-extensions-VS-Code
chmod +x scripts/setup.sh
./scripts/setup.sh
```

## 30-Second Setup (Windows)

```bash
git clone https://github.com/zombiecoder1/TypeScript-extensions-VS-Code.git
cd TypeScript-extensions-VS-Code
.\scripts\setup.bat
```

## First Run

1. Open VS Code
2. Go to Extensions > ZombieCoder
3. Click "Settings"
4. Add your API key (OpenAI, Anthropic, or use local Ollama)
5. Start chatting!
